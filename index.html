To Buy List

Application description

Here you got a to do list. You can dynamically add, edit and remove items by pressing the corresponding buttons. Click the item in order to mark it as complete. All items are automatically added, removed and edited in the LocalStorage so you can access your to do list after refresh the web padge. Try it yourself

Technologies used

  • HTML
  • CSS/SCSS
  • Vanilla JavaScript
  • Local Storage API
